Chi ha inventato questo linguaggio?

Html è stato inventato da Tom Bernera-lee.


Quando è stato inventato?

Nel 1989


Quali sono gli scopi di questo linguaggio?

Lo scopo di questo linguaggio html è descrivere la struttura e il contenuto delle pagine web.


Che cose il WEB?

Il web è un sistema di documenti interconnessitramite link,accessibili via internet.


Che problemi risolveva il web?

Ha risolto l'accesso all'informazione, la comunicazione, il commercio, la collaborazione e l'intrattenimento.


Che differenza tra web html ed internet?

Internet è una rete globale di computer interconnessi che utilizzano dei protocolli per comunicare, il web è un servizio che usa internet per accedere a documenti tramite il protocollo http mentre html un linguaggio informatico per creare le strutture delle pagine web.
